# Events Reference

Complete reference for capitan events emitted by sctx.

## Signals

### Token Lifecycle

| Signal | Name | Description |
|--------|------|-------------|
| `TokenGenerated` | `sctx.token.generated` | Security token created from certificate |
| `TokenVerified` | `sctx.token.verified` | Security token successfully verified |
| `TokenRejected` | `sctx.token.rejected` | Security token verification failed |

### Guard Lifecycle

| Signal | Name | Description |
|--------|------|-------------|
| `GuardCreated` | `sctx.guard.created` | Permission guard created by token holder |
| `GuardValidated` | `sctx.guard.validated` | Guard validation succeeded for token |
| `GuardRejected` | `sctx.guard.rejected` | Guard validation failed for token |

### Context Lifecycle

| Signal | Name | Description |
|--------|------|-------------|
| `ContextRevoked` | `sctx.context.revoked` | Security context manually revoked |

### Assertion Lifecycle

| Signal | Name | Description |
|--------|------|-------------|
| `AssertionValidated` | `sctx.assertion.validated` | Assertion validation succeeded |
| `AssertionRejected` | `sctx.assertion.rejected` | Assertion validation failed |

### Certificate Events

| Signal | Name | Description |
|--------|------|-------------|
| `CertificateRejected` | `sctx.certificate.rejected` | Certificate verification failed |

### Cache Operations

| Signal | Name | Description |
|--------|------|-------------|
| `CacheStored` | `sctx.cache.stored` | Security context stored in cache |
| `CacheHit` | `sctx.cache.hit` | Cache lookup found existing context |
| `CacheMiss` | `sctx.cache.miss` | Cache lookup did not find context |
| `CacheDeleted` | `sctx.cache.deleted` | Security context deleted from cache |
| `CacheExpired` | `sctx.cache.expired` | Security context expired during cleanup |

## Field Keys

### Identity Fields

```go
FingerprintKey = capitan.NewStringKey("fingerprint")
CommonNameKey  = capitan.NewStringKey("common_name")
```

| Key | Type | Description |
|-----|------|-------------|
| `FingerprintKey` | `string` | SHA-256 fingerprint of certificate (base64) |
| `CommonNameKey` | `string` | Certificate subject common name |

### Permission Fields

```go
PermissionsKey   = capitan.NewStringKey("permissions")
RequiredPermsKey = capitan.NewStringKey("required_permissions")
```

| Key | Type | Description |
|-----|------|-------------|
| `PermissionsKey` | `string` | Comma-separated list of granted permissions |
| `RequiredPermsKey` | `string` | Comma-separated list of required permissions |

### Guard Fields

```go
GuardIDKey = capitan.NewStringKey("guard_id")
```

| Key | Type | Description |
|-----|------|-------------|
| `GuardIDKey` | `string` | Unique guard identifier (hex) |

### Error Fields

```go
ErrorKey = capitan.NewStringKey("error")
```

| Key | Type | Description |
|-----|------|-------------|
| `ErrorKey` | `string` | Error message for rejection events |

### Time Fields

```go
ExpiresAtKey  = capitan.NewTimeKey("expires_at")
DurationMsKey = capitan.NewInt64Key("duration_ms")
```

| Key | Type | Description |
|-----|------|-------------|
| `ExpiresAtKey` | `time.Time` | Context expiration timestamp |
| `DurationMsKey` | `int64` | Operation duration in milliseconds |

## Usage Example

```go
import (
    "context"
    "github.com/zoobzio/capitan"
    "github.com/zoobzio/sctx"
)

// Hook specific signals
capitan.Hook(sctx.TokenGenerated, func(ctx context.Context, e *capitan.Event) {
    fingerprint, _ := sctx.FingerprintKey.From(e)
    commonName, _ := sctx.CommonNameKey.From(e)
    permissions, _ := sctx.PermissionsKey.From(e)

    log.Printf("Token generated for %s (fingerprint: %s) with permissions: %s",
        commonName, fingerprint[:16], permissions)
})

capitan.Hook(sctx.GuardRejected, func(ctx context.Context, e *capitan.Event) {
    guardID, _ := sctx.GuardIDKey.From(e)
    errorMsg, _ := sctx.ErrorKey.From(e)

    log.Printf("Guard %s rejected: %s", guardID, errorMsg)
})

// Observe multiple signals
capitan.Observe(func(ctx context.Context, e *capitan.Event) {
    // Handle all sctx events
},
    sctx.TokenGenerated,
    sctx.TokenRejected,
    sctx.GuardCreated,
    sctx.GuardRejected,
)
```

## Severity Levels

All sctx events use `capitan.SeverityInfo` by default. Filter by severity if needed:

```go
capitan.Observe(func(ctx context.Context, e *capitan.Event) {
    if e.Severity() == capitan.SeverityError {
        alerting.Send(e.Signal().Name(), extractFields(e))
    }
})
```
