# API Reference

Complete API reference for the `github.com/zoobzio/sctx` package.

## Admin Service

### NewAdminService

```go
func NewAdminService[M any](privateKey crypto.PrivateKey, trustedCAs *x509.CertPool) (Admin[M], error)
```

Creates the singleton admin service. Only one admin service can exist per application. Returns `ErrAdminAlreadyCreated` if called twice.

**Parameters:**
- `privateKey` - Private key for signing tokens (Ed25519 or ECDSA P-256)
- `trustedCAs` - Certificate pool of trusted CA certificates

### ResetAdminForTesting

```go
func ResetAdminForTesting()
```

Resets the admin singleton state. **Only available with the `testing` build tag.**

```bash
go test -tags=testing ./...
```

This restriction prevents the function from being available in production builds.

## Admin Interface

```go
type Admin[M any] interface {
    Generate(ctx context.Context, cert *x509.Certificate, assertion SignedAssertion) (SignedToken, error)
    CreateGuard(ctx context.Context, token SignedToken, requiredPermissions ...string) (Guard, error)
    SetGuardCreationPermissions(perms []string)
    SetPolicy(policy ContextPolicy[M]) error
    SetCache(cache ContextCache[M]) error
    RevokeByFingerprint(ctx context.Context, fingerprint string) error
    GetContext(ctx context.Context, fingerprint string) (*Context[M], bool)
    ActiveCount() int
}
```

### Generate

```go
func (a *Admin[M]) Generate(ctx context.Context, cert *x509.Certificate, assertion SignedAssertion) (SignedToken, error)
```

Generates a signed token for a certificate. Verifies the certificate chain, validates the assertion, applies the policy, caches the context, and returns a signed token.

### CreateGuard

```go
func (a *Admin[M]) CreateGuard(ctx context.Context, token SignedToken, requiredPermissions ...string) (Guard, error)
```

Creates a permission guard from a token. Fails if the token is invalid, expired, or lacks required permissions.

### SetGuardCreationPermissions

```go
func (a *Admin[M]) SetGuardCreationPermissions(perms []string)
```

Sets permissions required to create guards. When set, only tokens with these permissions can create guards.

### SetPolicy

```go
func (a *Admin[M]) SetPolicy(policy ContextPolicy[M]) error
```

Sets the context policy function. The policy transforms certificates into contexts.

### SetCache

```go
func (a *Admin[M]) SetCache(cache ContextCache[M]) error
```

Replaces the context cache implementation.

### RevokeByFingerprint

```go
func (a *Admin[M]) RevokeByFingerprint(ctx context.Context, fingerprint string) error
```

Revokes a context by certificate fingerprint. Removes from cache and emits `ContextRevoked` event.

### GetContext

```go
func (a *Admin[M]) GetContext(ctx context.Context, fingerprint string) (*Context[M], bool)
```

Retrieves a context from cache by fingerprint.

### ActiveCount

```go
func (a *Admin[M]) ActiveCount() int
```

Returns the number of active (non-expired) contexts in cache.


## Guard Interface

```go
type Guard interface {
    ID() string
    Validate(ctx context.Context, tokens ...SignedToken) error
    Permissions() []string
}
```

### ID

```go
func (g *Guard) ID() string
```

Returns the unique guard identifier.

### Validate

```go
func (g *Guard) Validate(ctx context.Context, tokens ...SignedToken) error
```

Validates one or more tokens against the guard's required permissions.

### Permissions

```go
func (g *Guard) Permissions() []string
```

Returns a copy of the required permissions.

## Tokens

### SignedToken

```go
type SignedToken string
```

Opaque, tamper-proof security token. Contains a signed reference to a cached context.

## Policy

### ContextPolicy

```go
type ContextPolicy[M any] func(cert *x509.Certificate) (*Context[M], error)
```

Function type for transforming certificates into contexts.

### DefaultContextPolicy

```go
func DefaultContextPolicy[M any]() ContextPolicy[M]
```

Returns the default policy that grants no permissions and sets 1-hour expiration.

## Cache

### ContextCache Interface

```go
type ContextCache[M any] interface {

}
```

### NewMemoryContextCache

```go
func NewMemoryContextCache[M any](cleanupInterval time.Duration) ContextCache[M]
```

Creates an in-memory cache with periodic cleanup.

## Context Guards

Utility functions for building context policies.

### RequireCertField

```go
func RequireCertField[M any](field, expected string) ContextGuard[M]
```

Returns a guard that requires a certificate field to match an expected value.

### RequireCertPattern

```go
func RequireCertPattern[M any](field, pattern string) ContextGuard[M]
```

Returns a guard that requires a certificate field to match a regex pattern.

### GrantPermissions

```go
func GrantPermissions[M any](permissions ...string) ContextGuard[M]
```

Returns a guard that adds permissions to a context.

### SetContext

```go
func SetContext[M any](opts ContextOptions) ContextGuard[M]
```

Returns a guard that sets multiple context fields at once.
