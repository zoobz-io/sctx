# Permission Guards

Guards enforce permission requirements on tokens. They're the primary mechanism for authorization checks in sctx.

## Creating Guards

Create a guard from a token with required permissions:

```go
guard, err := admin.CreateGuard(ctx, token, "read", "write")
if err != nil {
    // Token doesn't have required permissions to create a guard
    // or the token itself is invalid
}
```

Guard creation fails if:
- The token is invalid or expired
- The token's context doesn't have the required permissions
- Guard creation permissions are set and the token lacks them

## Validating Tokens

Use guards to validate tokens:

```go
err := guard.Validate(ctx, token)
if err != nil {
    // Token validation failed
    log.Printf("Access denied: %v", err)
    return
}
// Token is valid and has required permissions
```

Validation checks:
1. Token signature is valid
2. Context hasn't expired
3. Context hasn't been revoked
4. Context has all required permissions

## Guard Methods

```go
guard.ID()          // Unique guard identifier
guard.Permissions() // Returns a copy of required permissions
guard.Validate(ctx, tokens...)  // Validate one or more tokens
```

## Permission Patterns

### Resource-Based Permissions

```go
// Create guards for specific resources
readUsersGuard, _ := admin.CreateGuard(ctx, token, "users:read")
writeUsersGuard, _ := admin.CreateGuard(ctx, token, "users:write")
deleteUsersGuard, _ := admin.CreateGuard(ctx, token, "users:delete")

func handleGetUser(w http.ResponseWriter, r *http.Request) {
    if err := readUsersGuard.Validate(r.Context(), getToken(r)); err != nil {
        http.Error(w, "Forbidden", http.StatusForbidden)
        return
    }
    // ... handle request
}
```

### Wildcard Permissions

Implement wildcards in your policy:

```go
admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[any], error) {
    var perms []string

    // Admin gets wildcard
    if isAdmin(cert) {
        perms = append(perms, "users:*", "orders:*", "admin:*")
    }

    return &sctx.Context[any]{
        Permissions: perms,
        ExpiresAt:   time.Now().Add(time.Hour),
    }, nil
})

// In validation, check for exact match or wildcard
func hasPermission(ctx *sctx.Context[any], required string) bool {
    for _, p := range ctx.Permissions {
        if p == required {
            return true
        }
        // Check wildcard
        if strings.HasSuffix(p, ":*") {
            prefix := strings.TrimSuffix(p, "*")
            if strings.HasPrefix(required, prefix) {
                return true
            }
        }
    }
    return false
}
```

### Multiple Permission Requirements

Guards require all specified permissions:

```go
// Token must have BOTH read AND write
guard, err := admin.CreateGuard(ctx, token, "read", "write")
```

For OR logic, create separate guards:

```go
readGuard, _ := admin.CreateGuard(ctx, token, "read")
adminGuard, _ := admin.CreateGuard(ctx, token, "admin")

// Token can have read OR admin
if readGuard.Validate(ctx, token) != nil && adminGuard.Validate(ctx, token) != nil {
    http.Error(w, "Forbidden", http.StatusForbidden)
    return
}
```

## Guard Creation Permissions

Restrict who can create guards:

```go
// Only tokens with "guard:create" permission can create guards
admin.SetGuardCreationPermissions([]string{"guard:create"})

// Now this fails without "guard:create" permission
guard, err := admin.CreateGuard(ctx, limitedToken, "read")
// err != nil if limitedToken lacks "guard:create"
```

## HTTP Middleware

Create middleware with guards:

```go
func RequirePermissions(admin sctx.Admin[any], perms ...string) func(http.Handler) http.Handler {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            token := extractToken(r)
            if token == "" {
                http.Error(w, "Unauthorized", http.StatusUnauthorized)
                return
            }

            guard, err := admin.CreateGuard(r.Context(), sctx.SignedToken(token), perms...)
            if err != nil {
                http.Error(w, "Forbidden", http.StatusForbidden)
                return
            }

            if err := guard.Validate(r.Context(), sctx.SignedToken(token)); err != nil {
                http.Error(w, "Forbidden", http.StatusForbidden)
                return
            }

            next.ServeHTTP(w, r)
        })
    }
}

// Usage
mux.Handle("/api/users", RequirePermissions(admin, "users:read")(usersHandler))
mux.Handle("/api/admin", RequirePermissions(admin, "admin:access")(adminHandler))
```

## gRPC Interceptor

```go
func UnaryAuthInterceptor(admin sctx.Admin[any]) grpc.UnaryServerInterceptor {
    return func(ctx context.Context, req any, info *grpc.UnaryServerInfo, handler grpc.UnaryHandler) (any, error) {
        // Extract token from metadata
        md, ok := metadata.FromIncomingContext(ctx)
        if !ok {
            return nil, status.Error(codes.Unauthenticated, "missing metadata")
        }

        tokens := md.Get("authorization")
        if len(tokens) == 0 {
            return nil, status.Error(codes.Unauthenticated, "missing token")
        }

        token := sctx.SignedToken(strings.TrimPrefix(tokens[0], "Bearer "))

        // Get required permissions for this method
        perms := methodPermissions[info.FullMethod]

        guard, err := admin.CreateGuard(ctx, token, perms...)
        if err != nil {
            return nil, status.Error(codes.PermissionDenied, "insufficient permissions")
        }

        if err := guard.Validate(ctx, token); err != nil {
            return nil, status.Error(codes.PermissionDenied, err.Error())
        }

        return handler(ctx, req)
    }
}
```

## Caching Guards

Guards are lightweight, but for high-throughput scenarios, cache them:

```go
type GuardCache struct {
    mu     sync.RWMutex
    guards map[string]sctx.Guard
}

func (c *GuardCache) GetOrCreate(admin sctx.Admin[any], ctx context.Context, token sctx.SignedToken, perms ...string) (sctx.Guard, error) {
    key := fmt.Sprintf("%s:%s", token, strings.Join(perms, ","))

    c.mu.RLock()
    if guard, ok := c.guards[key]; ok {
        c.mu.RUnlock()
        return guard, nil
    }
    c.mu.RUnlock()

    guard, err := admin.CreateGuard(ctx, token, perms...)
    if err != nil {
        return nil, err
    }

    c.mu.Lock()
    c.guards[key] = guard
    c.mu.Unlock()

    return guard, nil
}
```

## Guard Events

All guard operations emit capitan events:

```go
// Listen for guard creation
capitan.Hook(sctx.GuardCreated, func(ctx context.Context, e *capitan.Event) {
    guardID, _ := sctx.GuardIDKey.From(e)
    fingerprint, _ := sctx.FingerprintKey.From(e)
    perms, _ := sctx.RequiredPermsKey.From(e)
    log.Printf("Guard %s created for %s with perms: %s", guardID, fingerprint, perms)
})

// Listen for validation failures
capitan.Hook(sctx.GuardRejected, func(ctx context.Context, e *capitan.Event) {
    guardID, _ := sctx.GuardIDKey.From(e)
    err, _ := sctx.ErrorKey.From(e)
    log.Printf("Guard %s rejected: %s", guardID, err)
})
```
