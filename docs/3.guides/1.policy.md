# Context Policies

A context policy defines how X.509 certificates are transformed into security contexts. Policies are the bridge between your PKI and your authorization model.

## Basic Policy

Set a policy on the admin service:

```go
admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[any], error) {
    return &sctx.Context[any]{
        Permissions: []string{"read"},
        ExpiresAt:   time.Now().Add(time.Hour),
    }, nil
})
```

The policy receives the verified certificate and returns a context or an error.

## Extracting Certificate Fields

Use certificate fields to drive authorization decisions:

```go
admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[UserMeta], error) {
    // Extract common name for identity
    identity := cert.Subject.CommonName

    // Extract organization for tenant isolation
    var tenantID string
    if len(cert.Subject.Organization) > 0 {
        tenantID = cert.Subject.Organization[0]
    }

    // Use organizational units for role mapping
    var perms []string
    for _, ou := range cert.Subject.OrganizationalUnit {
        switch ou {
        case "Administrators":
            perms = append(perms, "admin:*")
        case "Developers":
            perms = append(perms, "code:read", "code:write")
        case "Auditors":
            perms = append(perms, "audit:read")
        }
    }

    return &sctx.Context[UserMeta]{
        Permissions: perms,
        ExpiresAt:   time.Now().Add(time.Hour),
        Metadata: UserMeta{
            Identity: identity,
            TenantID: tenantID,
        },
    }, nil
})
```

## Rejecting Certificates

Return an error to reject a certificate:

```go
admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[any], error) {
    // Reject expired certificates
    if time.Now().After(cert.NotAfter) {
        return nil, errors.New("certificate has expired")
    }

    // Reject certificates without required extension
    if !hasRequiredExtension(cert) {
        return nil, errors.New("certificate missing required extension")
    }

    // Reject certificates from untrusted organizations
    if len(cert.Subject.Organization) == 0 {
        return nil, errors.New("certificate must have organization")
    }

    allowedOrgs := map[string]bool{
        "Acme Corp":    true,
        "Partner Inc":  true,
    }
    if !allowedOrgs[cert.Subject.Organization[0]] {
        return nil, errors.New("organization not authorized")
    }

    return &sctx.Context[any]{
        Permissions: []string{"read"},
        ExpiresAt:   time.Now().Add(time.Hour),
    }, nil
})
```

## Dynamic Expiration

Set expiration based on certificate properties:

```go
admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[any], error) {
    // Context expires when certificate expires, or in 1 hour, whichever is sooner
    expiry := time.Now().Add(time.Hour)
    if cert.NotAfter.Before(expiry) {
        expiry = cert.NotAfter
    }

    // Shorter expiry for sensitive operations
    if containsSensitiveOU(cert) {
        expiry = time.Now().Add(15 * time.Minute)
    }

    return &sctx.Context[any]{
        Permissions: []string{"read"},
        ExpiresAt:   expiry,
    }, nil
})
```

## Permission Hierarchies

Build permission hierarchies from certificate structure:

```go
func permissionsFromCert(cert *x509.Certificate) []string {
    var perms []string

    // Base permissions for all authenticated users
    perms = append(perms, "api:access")

    // Organization-level permissions
    for _, org := range cert.Subject.Organization {
        perms = append(perms, fmt.Sprintf("org:%s:read", org))
    }

    // Role-based permissions from OUs
    for _, ou := range cert.Subject.OrganizationalUnit {
        rolePerms := rolePermissionMap[ou]
        perms = append(perms, rolePerms...)
    }

    // DNS SANs grant service access
    for _, dns := range cert.DNSNames {
        if strings.HasSuffix(dns, ".internal.company.com") {
            perms = append(perms, "internal:access")
            break
        }
    }

    return perms
}

var rolePermissionMap = map[string][]string{
    "Engineering":   {"code:read", "code:write", "deploy:staging"},
    "Operations":    {"infra:read", "infra:write", "deploy:*"},
    "Security":      {"audit:*", "secrets:read"},
    "Executives":    {"reports:*", "billing:read"},
}
```

## Metadata Population

Use the generic metadata field for application-specific data:

```go
type ServiceMeta struct {
    ServiceName string
    Environment string
    Version     string
    Capabilities []string
}

admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[ServiceMeta], error) {
    // Parse service metadata from certificate
    meta := ServiceMeta{
        ServiceName: cert.Subject.CommonName,
    }

    // Extract environment from OU
    for _, ou := range cert.Subject.OrganizationalUnit {
        if strings.HasPrefix(ou, "env:") {
            meta.Environment = strings.TrimPrefix(ou, "env:")
        }
        if strings.HasPrefix(ou, "v") {
            meta.Version = ou
        }
    }

    // Extract capabilities from custom extension
    for _, ext := range cert.Extensions {
        if ext.Id.Equal(customCapabilitiesOID) {
            meta.Capabilities = parseCapabilities(ext.Value)
        }
    }

    return &sctx.Context[ServiceMeta]{
        Metadata:    meta,
        Permissions: capabilitiesToPermissions(meta.Capabilities),
        ExpiresAt:   time.Now().Add(time.Hour),
    }, nil
})
```

## External Authorization

Call external services for authorization decisions:

```go
admin.SetPolicy(func(cert *x509.Certificate) (*sctx.Context[any], error) {
    // Call authorization service
    ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
    defer cancel()

    authResp, err := authService.GetPermissions(ctx, &authpb.Request{
        CertificateFingerprint: sctx.GetFingerprint(cert),
        CommonName:             cert.Subject.CommonName,
        Organization:           cert.Subject.Organization,
    })
    if err != nil {
        return nil, fmt.Errorf("authorization service error: %w", err)
    }

    if !authResp.Authorized {
        return nil, errors.New("not authorized")
    }

    return &sctx.Context[any]{
        Permissions: authResp.Permissions,
        ExpiresAt:   time.Now().Add(time.Duration(authResp.TtlSeconds) * time.Second),
    }, nil
})
```

## Default Policy

If no policy is set, sctx uses a default policy that:
- Grants no permissions
- Sets 1-hour expiration
- Extracts certificate info

```go
func DefaultContextPolicy[M any]() ContextPolicy[M] {
    return func(cert *x509.Certificate) (*Context[M], error) {
        if cert == nil {
            return nil, errors.New("certificate required")
        }
        return &Context[M]{
            CertificateFingerprint: GetFingerprint(cert),
            CertificateInfo:        extractCertificateInfo(cert),
            IssuedAt:               time.Now(),
            ExpiresAt:              time.Now().Add(time.Hour),
            Permissions:            []string{},
        }, nil
    }
}
```

## Policy Testing

Test policies with generated certificates:

```go
func TestPolicy(t *testing.T) {
    // Generate test certificate with specific attributes
    cert := generateTestCert(t, func(template *x509.Certificate) {
        template.Subject.Organization = []string{"Acme Corp"}
        template.Subject.OrganizationalUnit = []string{"Engineering"}
    })

    policy := myPolicy()
    ctx, err := policy(cert)

    require.NoError(t, err)
    assert.Contains(t, ctx.Permissions, "code:read")
    assert.Equal(t, "Acme Corp", ctx.Metadata.TenantID)
}
```
