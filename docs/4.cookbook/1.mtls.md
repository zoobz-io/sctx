# mTLS Integration

*Recipe: Use sctx with your existing mTLS setup.*

sctx is designed to work with mutual TLS. If you're already doing mTLS, you have certificates - sctx turns them into typed security contexts.

## The Scenario

You have an HTTP server configured for mTLS. Clients present certificates during the TLS handshake. You want to:
- Extract security contexts from those certificates
- Generate tokens for authenticated sessions
- Authorize requests with permission guards

## Server Setup

Configure your HTTP server for mTLS:

```go
package main

import (
    "crypto/tls"
    "crypto/x509"
    "log"
    "net/http"
    "os"

    "github.com/zoobzio/sctx"
)

func main() {
    // Load CA certificate pool
    caCert, _ := os.ReadFile("ca.crt")
    certPool := x509.NewCertPool()
    certPool.AppendCertsFromPEM(caCert)

    // Load server key for token signing
    serverKey, _ := loadPrivateKey("server.key")

    // Create sctx admin
    admin, _ := sctx.NewAdminService[UserMeta](serverKey, certPool)

    // Set authorization policy
    admin.SetPolicy(myPolicy)

    // Configure TLS
    tlsConfig := &tls.Config{
        ClientAuth: tls.RequireAndVerifyClientCert,
        ClientCAs:  certPool,
        MinVersion: tls.VersionTLS13,
    }

    server := &http.Server{
        Addr:      ":8443",
        TLSConfig: tlsConfig,
        Handler:   newRouter(admin),
    }

    log.Fatal(server.ListenAndServeTLS("server.crt", "server.key"))
}
```

## Token Generation Endpoint

Create an endpoint that generates tokens from mTLS certificates:

```go
func tokenHandler(admin sctx.Admin[UserMeta]) http.HandlerFunc {
    return func(w http.ResponseWriter, r *http.Request) {
        // Get client certificate from TLS connection
        if r.TLS == nil || len(r.TLS.PeerCertificates) == 0 {
            http.Error(w, "Client certificate required", http.StatusUnauthorized)
            return
        }
        clientCert := r.TLS.PeerCertificates[0]

        // Client sends assertion in request body
        var req struct {
            Assertion sctx.SignedAssertion `json:"assertion"`
        }
        if err := json.NewDecoder(r.Body).Decode(&req); err != nil {
            http.Error(w, "Invalid request", http.StatusBadRequest)
            return
        }

        // Generate token
        token, err := admin.Generate(r.Context(), clientCert, req.Assertion)
        if err != nil {
            http.Error(w, err.Error(), http.StatusForbidden)
            return
        }

        // Return token
        json.NewEncoder(w).Encode(map[string]string{
            "token": string(token),
        })
    }
}
```

## Client Implementation

Client creates assertion and requests token:

```go
func getToken(clientCert *x509.Certificate, clientKey crypto.PrivateKey, serverURL string) (sctx.SignedToken, error) {
    // Create assertion proving key ownership
    assertion, err := sctx.CreateAssertion(clientKey, clientCert)
    if err != nil {
        return "", err
    }

    // Prepare request
    body, _ := json.Marshal(map[string]any{
        "assertion": assertion,
    })

    req, _ := http.NewRequest("POST", serverURL+"/token", bytes.NewReader(body))

    // Configure TLS with client certificate
    tlsConfig := &tls.Config{
        Certificates: []tls.Certificate{
            {
                Certificate: [][]byte{clientCert.Raw},
                PrivateKey:  clientKey,
            },
        },
    }
    client := &http.Client{
        Transport: &http.Transport{TLSClientConfig: tlsConfig},
    }

    // Make request
    resp, err := client.Do(req)
    if err != nil {
        return "", err
    }
    defer resp.Body.Close()

    // Parse response
    var result struct {
        Token string `json:"token"`
    }
    json.NewDecoder(resp.Body).Decode(&result)

    return sctx.SignedToken(result.Token), nil
}
```

## Authorization Middleware

Middleware that validates tokens:

```go
func authMiddleware(admin sctx.Admin[UserMeta], requiredPerms ...string) func(http.Handler) http.Handler {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            // Extract token from header
            authHeader := r.Header.Get("Authorization")
            if authHeader == "" || !strings.HasPrefix(authHeader, "Bearer ") {
                http.Error(w, "Missing authorization", http.StatusUnauthorized)
                return
            }
            token := sctx.SignedToken(strings.TrimPrefix(authHeader, "Bearer "))

            // Create guard and validate
            guard, err := admin.CreateGuard(r.Context(), token, requiredPerms...)
            if err != nil {
                http.Error(w, "Insufficient permissions", http.StatusForbidden)
                return
            }

            if err := guard.Validate(r.Context(), token); err != nil {
                http.Error(w, err.Error(), http.StatusForbidden)
                return
            }

            next.ServeHTTP(w, r)
        })
    }
}
```

## Complete Router

```go
func newRouter(admin sctx.Admin[UserMeta]) http.Handler {
    mux := http.NewServeMux()

    // Token endpoint - requires mTLS but no token
    mux.HandleFunc("POST /token", tokenHandler(admin))

    // Protected endpoints - require token with permissions
    mux.Handle("GET /api/users",
        authMiddleware(admin, "users:read")(usersHandler()))
    mux.Handle("POST /api/users",
        authMiddleware(admin, "users:write")(createUserHandler()))
    mux.Handle("DELETE /api/users/{id}",
        authMiddleware(admin, "users:delete")(deleteUserHandler()))

    // Admin endpoints
    mux.Handle("GET /admin/contexts",
        authMiddleware(admin, "admin:contexts")(contextsHandler(admin)))

    return mux
}
```

## Policy Based on Certificate

Map certificate attributes to permissions:

```go
type UserMeta struct {
    Identity string
    Tenant   string
    Role     string
}

func myPolicy(cert *x509.Certificate) (*sctx.Context[UserMeta], error) {
    var perms []string

    // Map OUs to permissions
    for _, ou := range cert.Subject.OrganizationalUnit {
        switch ou {
        case "Administrators":
            perms = append(perms, "admin:*", "users:*")
        case "Developers":
            perms = append(perms, "users:read", "api:access")
        case "Auditors":
            perms = append(perms, "users:read", "audit:read")
        }
    }

    // Extract tenant from organization
    var tenant string
    if len(cert.Subject.Organization) > 0 {
        tenant = cert.Subject.Organization[0]
    }

    return &sctx.Context[UserMeta]{
        Permissions: perms,
        ExpiresAt:   time.Now().Add(time.Hour),
        Metadata: UserMeta{
            Identity: cert.Subject.CommonName,
            Tenant:   tenant,
            Role:     determineRole(cert),
        },
    }, nil
}
```

## Token Refresh

Implement token refresh before expiration:

```go
func refreshHandler(admin sctx.Admin[UserMeta]) http.HandlerFunc {
    return func(w http.ResponseWriter, r *http.Request) {
        // Verify current token is still valid
        oldToken := sctx.SignedToken(strings.TrimPrefix(
            r.Header.Get("Authorization"), "Bearer "))

        // Get client cert from mTLS
        clientCert := r.TLS.PeerCertificates[0]

        // Create new assertion
        var req struct {
            Assertion sctx.SignedAssertion `json:"assertion"`
        }
        json.NewDecoder(r.Body).Decode(&req)

        // Generate new token
        newToken, err := admin.Generate(r.Context(), clientCert, req.Assertion)
        if err != nil {
            http.Error(w, err.Error(), http.StatusForbidden)
            return
        }

        json.NewEncoder(w).Encode(map[string]string{
            "token": string(newToken),
        })
    }
}
```

## What This Produces

1. Clients connect via mTLS with their certificates
2. Clients create assertions proving key ownership
3. Server generates signed tokens from certificate + assertion
4. Clients include tokens in subsequent requests
5. Guards validate tokens and check permissions

The mTLS handshake provides authentication. sctx provides authorization.
