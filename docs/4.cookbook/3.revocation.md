# Context Revocation

*Recipe: Immediately revoke security contexts when needed.*

## The Scenario

You need to revoke access immediately when:
- A user leaves the organization
- A certificate is compromised
- A security incident is detected
- Administrative action is required

sctx provides instant revocation that takes effect immediately - no waiting for token expiration.

## Basic Revocation

Revoke a context by certificate fingerprint:

```go
func revokeUser(admin sctx.Admin[any], cert *x509.Certificate) error {
    fingerprint := sctx.GetFingerprint(cert)
    return admin.RevokeByFingerprint(context.Background(), fingerprint)
}
```

After revocation:
- Existing guards immediately reject the token
- New guards cannot be created for the token
- The context is removed from cache
- A `ContextRevoked` event is emitted

## Revocation Endpoint

Create an admin endpoint for revocation:

```go
func revokeHandler(admin sctx.Admin[any]) http.HandlerFunc {
    return func(w http.ResponseWriter, r *http.Request) {
        var req struct {
            Fingerprint string `json:"fingerprint"`
            Reason      string `json:"reason"`
        }
        if err := json.NewDecoder(r.Body).Decode(&req); err != nil {
            http.Error(w, "Invalid request", http.StatusBadRequest)
            return
        }

        // Log revocation reason
        log.Printf("Revoking context %s: %s", req.Fingerprint[:16], req.Reason)

        // Revoke
        if err := admin.RevokeByFingerprint(r.Context(), req.Fingerprint); err != nil {
            http.Error(w, err.Error(), http.StatusInternalServerError)
            return
        }

        w.WriteHeader(http.StatusNoContent)
    }
}
```

## Bulk Revocation

Revoke multiple contexts at once:

```go
func revokeMultiple(admin sctx.Admin[any], fingerprints []string) []error {
    var errors []error
    ctx := context.Background()

    for _, fp := range fingerprints {
        if err := admin.RevokeByFingerprint(ctx, fp); err != nil {
            errors = append(errors, fmt.Errorf("failed to revoke %s: %w", fp[:16], err))
        }
    }

    return errors
}
```

## Organization-Wide Revocation

Revoke all contexts for an organization:

```go
type ContextStore interface {
    GetFingerprintsByOrg(ctx context.Context, org string) ([]string, error)
}

func revokeOrganization(admin sctx.Admin[any], store ContextStore, org string) error {
    ctx := context.Background()

    // Get all fingerprints for the organization
    fingerprints, err := store.GetFingerprintsByOrg(ctx, org)
    if err != nil {
        return err
    }

    // Revoke each
    var revokeErrors []error
    for _, fp := range fingerprints {
        if err := admin.RevokeByFingerprint(ctx, fp); err != nil {
            revokeErrors = append(revokeErrors, err)
        }
    }

    if len(revokeErrors) > 0 {
        return fmt.Errorf("failed to revoke %d contexts", len(revokeErrors))
    }

    return nil
}
```

## Revocation with Notification

Notify affected users when their context is revoked:

```go
func revokeWithNotification(
    admin sctx.Admin[UserMeta],
    notifier Notifier,
    fingerprint string,
    reason string,
) error {
    ctx := context.Background()

    // Get context before revocation to access metadata
    sctxCtx, ok := admin.GetContext(ctx, fingerprint)
    if !ok {
        // Already revoked or doesn't exist
        return nil
    }

    // Revoke
    if err := admin.RevokeByFingerprint(ctx, fingerprint); err != nil {
        return err
    }

    // Notify user
    notifier.Notify(sctxCtx.Metadata.Email, NotificationMessage{
        Subject: "Access Revoked",
        Body:    fmt.Sprintf("Your access has been revoked. Reason: %s", reason),
    })

    return nil
}
```

## Scheduled Revocation

Revoke contexts on a schedule (e.g., when certificates expire):

```go
func startRevocationScheduler(admin sctx.Admin[any], store ContextStore) {
    ticker := time.NewTicker(1 * time.Minute)
    go func() {
        for range ticker.C {
            revokeExpiredContexts(admin, store)
        }
    }()
}

func revokeExpiredContexts(admin sctx.Admin[any], store ContextStore) {
    ctx := context.Background()

    // Get fingerprints with expired certificates
    expired, err := store.GetExpiredCertFingerprints(ctx)
    if err != nil {
        log.Printf("Failed to get expired fingerprints: %v", err)
        return
    }

    for _, fp := range expired {
        if err := admin.RevokeByFingerprint(ctx, fp); err != nil {
            log.Printf("Failed to revoke expired context %s: %v", fp[:16], err)
        }
    }
}
```

## Revocation Events

Listen for revocation events:

```go
func setupRevocationAuditing() {
    capitan.Hook(sctx.ContextRevoked, func(ctx context.Context, e *capitan.Event) {
        fingerprint, _ := sctx.FingerprintKey.From(e)

        // Log to audit trail
        auditLog.Record(AuditEntry{
            Timestamp:   e.Timestamp(),
            Action:      "context_revoked",
            Fingerprint: fingerprint,
        })

        // Alert security team
        securityAlert.Send("Context Revoked", map[string]any{
            "fingerprint": fingerprint,
            "timestamp":   e.Timestamp(),
        })
    })
}
```

## Revocation Verification

Guards automatically check for revocation, but you can also check directly:

```go
func isRevoked(admin sctx.Admin[any], fingerprint string) bool {
    _, exists := admin.GetContext(context.Background(), fingerprint)
    return !exists
}
```

## Re-authentication After Revocation

Users can re-authenticate after revocation with a new assertion:

```go
func handleReauth(admin sctx.Admin[any], w http.ResponseWriter, r *http.Request) {
    cert := r.TLS.PeerCertificates[0]

    // Parse assertion from request
    var req struct {
        Assertion sctx.SignedAssertion `json:"assertion"`
    }
    json.NewDecoder(r.Body).Decode(&req)

    // Generate new token (creates new context)
    token, err := admin.Generate(r.Context(), cert, req.Assertion)
    if err != nil {
        http.Error(w, err.Error(), http.StatusForbidden)
        return
    }

    json.NewEncoder(w).Encode(map[string]string{
        "token": string(token),
    })
}
```

## Emergency Shutdown

Revoke all active contexts in an emergency:

```go
func emergencyShutdown(admin sctx.Admin[any], store ContextStore) error {
    ctx := context.Background()

    // Get all active fingerprints
    fingerprints, err := store.GetAllActiveFingerprints(ctx)
    if err != nil {
        return err
    }

    log.Printf("Emergency shutdown: revoking %d contexts", len(fingerprints))

    // Revoke all
    for _, fp := range fingerprints {
        _ = admin.RevokeByFingerprint(ctx, fp) // Best effort
    }

    return nil
}
```

## What This Produces

1. Instant revocation - guards fail immediately
2. Event emission for audit trails
3. Notification capabilities for user communication
4. Support for bulk and organizational revocation
5. Clean re-authentication path after revocation
